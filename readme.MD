## 🦅 Sobre o Projeto

O **Law Match** é uma aplicação Full Stack desenvolvida para transformar a leitura cansativa de códigos e artigos em uma experiência dinâmica e viciante. Inspirado na mecânica de "swipe", o usuário decide se domina ou não artigos fundamentais da legislação brasileira.

Este é o primeiro projeto oficial do ecossistema **Falkon**, focado em entregar soluções reais com interface de alto nível.

## ✨ Funcionalidades

- **Swipe de Artigos:** Decida entre "Conheço" ou "Não Conheço" para artigos do Código Civil e Penal.
- **Identidade Visual Premium:** Design em Dark Mode com a paleta exclusiva Falkon (Laranja Dourado e Preto).
- **Gamificação:** Sistema de pontuação e rodadas para manter o engajamento no estudo.
- **Responsividade Total:** Pronto para rodar no PC ou como Web App no celular.

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído com o que há de mais moderno no ecossistema Web:

- **[Vite](https://vitejs.dev/):** Build tool ultrarrápida.
- **[React](https://reactjs.org/):** Biblioteca para interfaces dinâmicas.
- **[Tailwind CSS](https://tailwindcss.com/):** Estilização de alta performance.
- **[Lucide React](https://lucide.dev/):** Ícones modernos e limpos.
